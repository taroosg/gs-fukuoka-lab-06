## 本講義の内容とゴール

### 本講義の内容

- 関数を定義し，動かす一連の動作を実装する．
- オンライン上のデータベースにデータを保存する．
- オンライン上のデータを取得し，画面上にリアルタイムに表示する．

### 本講義のゴール

- 関数を使って処理を実装できるようになる！
- オンライン&リアルタイムでデータ共有する！
- 複雑なデータを扱う場合にどう考えるかを把握する！
