# PHP基礎


## 本講義の内容とゴール

### 本講義の内容

- Webアプリケーションの仕組み．
- PHPの基本的な開発手順．
- PHPを用いた簡単な処理の実装．


### 本講義のゴール

- Webアプリケーションの仕組みをと動きを把握する（大事）！
- PHP（の開発手順）に慣れる！
- PHPを用いたデータの送受信を通して，アプリケーションの動きをイメージする．


## 準備

以下3点ができているか確認しよう！

- XAMPPの起動確認
- [http://localhost/](http://localhost/)のアクセス確認
- サンプルフォルダを「htdocs」フォルダに入れる
