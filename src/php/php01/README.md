# PHP 基礎

## 本講義の内容とゴール

### 本講義の内容

- Web アプリケーションの仕組み．
- PHP の基本的な開発手順．
- PHP を用いた簡単な処理の実装．

### 本講義のゴール

- Web アプリケーションの仕組みをと動きを把握する（大事）！
- PHP（の開発手順）に慣れる！
- PHP を用いたデータの送受信を通して，アプリケーションの動きをイメージする．

## 準備

以下 3 点ができているか確認しよう！

- XAMPP の起動確認
- [http://localhost/](http://localhost/)のアクセス確認
- サンプルフォルダを「htdocs」フォルダに入れる
